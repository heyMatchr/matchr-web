# Matchr

Matchr is a private mobile-first social matching platform built with Next.js and Supabase.

Designed around realtime interaction, premium dark UI, and responsive mobile UX, Matchr includes onboarding, profile discovery, mutual matching, and live messaging.

---

## Features

- Secure authentication
- Profile onboarding
- Discover feed
- Mutual matching system
- Realtime messaging
- Presence indicators
- Typing indicators
- Unread message tracking
- Mobile-first responsive navigation
- Supabase realtime backend
- Dark premium UI system

---

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Supabase Realtime
